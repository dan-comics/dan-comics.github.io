## âš™ï¸ How It Works

The website uses a **manifest.json** file generated by `build-manifest.py`:
- The script scans all folders in `Website/` directory
- Finds all image/PDF files in each folder and subfolder
- Generates `manifest.json` with the complete comic library
- The website loads from the manifest (no API calls = no rate limits!)

## ğŸ”§ Build Script Details

The `build-manifest.py` script:
- Automatically discovers all series folders
- Finds all comics in each folder
- Handles nested subfolders (like FLL - Band of Bots)
- Excludes Background.JPG and .DS_Store files
- Shows a summary of comics found

**Always run the build script after adding new comics!**

## ğŸ”§ Troubleshooting

**Comics not showing up?**
1. Did you run `python3 build-manifest.py`?
2. Check that `manifest.json` was updated
3. Did you commit and push `manifest.json`?
4. Wait 2-3 minutes for GitHub Pages
5. Clear your browser cache and refresh

**Build script errors?**
1. Make sure Python 3 is installed: `python3 --version`
2. Check that you're in the project directory
3. Ensure the `Website/` folder exists

**New series not appearing?**
1. Make sure the folder is inside `Website/`
2. Add at least one image file to the folder
3. Run the build script
4. Check that manifest.json includes the new series
